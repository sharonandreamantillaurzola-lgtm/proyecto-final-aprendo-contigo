Proyecto final del programa Jóvenes Creativos, en el cual se desarrolla una página web para el emprendimiento educativo “Aprendo Contigo”. El sitio tiene como objetivo presentar una plataforma de refuerzo académico personalizado para estudiantes de preescolar a octavo grado, ofreciendo información sobre el proyecto, sus servicios, beneficios, misión, visión y valores. Este proyecto busca aplicar conocimientos de desarrollo web con HTML, organizando una estructura clara, semántica y funcional como base para futuras mejoras en diseño y programación.

📌 Commit 1 - Estructura inicial del proyecto
En este primer commit se realizó la creación de la estructura base del proyecto web “Aprendo Contigo”.

Se desarrolló el archivo index.html con la estructura principal del sitio, incluyendo el encabezado (Header) con el nombre del proyecto y el menú de navegación, la sección principal (Hero) con la presentación del servicio y botones de acción, la sección de beneficios, y la sección “¿Quiénes somos?”, donde se describe el proyecto y su fundadora.

También se implementaron las secciones de misión, visión y valores del emprendimiento.

Adicionalmente, se organizó el proyecto creando carpetas para una mejor estructura:

Carpeta de imágenes (img)
Archivo de estilos (css)
Archivo de scripts (js)
# 📌 Commit 3 - Diseño visual y estilos CSS del proyecto

En este tercer commit se desarrolló completamente el diseño visual del sitio web **"Aprendo Contigo"** mediante CSS, aplicando estilos modernos, colores de marca y una mejor experiencia de usuario.

Se implementaron:

* Diseño del encabezado, menú de navegación y sección principal (Hero).
* Estilos personalizados para todas las secciones de la página.
* Tarjetas con sombras, bordes degradados, animaciones y efectos hover.
* Diseño de la sección **Quiénes somos**, áreas académicas, catálogo de servicios y testimonios.
* Aplicación de la tipografía Poppins y una identidad visual basada en los colores morado, azul y verde.
* Fondo personalizado con detalles visuales y marca de agua del logo.
* Adaptación responsive para diferentes dispositivos.

Con este commit se finalizó la parte visual del proyecto, dejando la página con un diseño completo, atractivo y organizado.
