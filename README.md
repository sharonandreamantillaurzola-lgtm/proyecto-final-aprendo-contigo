Proyecto final del programa Jóvenes Creativos, en el cual se desarrolla una página web para el emprendimiento educativo “Aprendo Contigo”.
El sitio tiene como objetivo presentar una plataforma de refuerzo académico personalizado para estudiantes de preescolar a octavo grado, ofreciendo información sobre el proyecto, sus servicios, beneficios, misión, visión y valores.
Este proyecto busca aplicar conocimientos de desarrollo web con HTML, organizando una estructura clara, semántica y funcional como base para futuras mejoras en diseño y programación.
